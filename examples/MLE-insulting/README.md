## 问题来源
本问题来源于MLE-Bench，选自Kaggle竞赛题目Detecting Insults in Social Commentary

## 问题描述
社交评论侮辱性检测的核心任务是基于提供的社交评论文本数据，预测其标签为“侮辱（insult）”或“非侮辱（non-insult）”，属于经典的文本分类问题。

## 应用场景
社交评论侮辱性检测在多个实际领域具有重要的应用价值：
* 内容安全与社区治理：社交媒体、论坛与评论区可利用此类模型自动识别侮辱性内容，减少人工审核压力，提升社区健康度。
* 舆情监测与品牌保护：企业可通过文本检测系统识别负面或攻击性言论，辅助品牌形象与公共关系管理。


## 伐谋成果
MLE-Bench 是一个由 OpenAI 提出的用于评估 AI  agent在真实机器学习工程任务中综合能力的基准数据集，该基准涵盖 75 个机器学习工程任务，考察 Agent 在数据处理、模型训练与实验执行方面的综合能力。伐谋 Agent 在 MLE-Bench 上得牌率为 43.56%，整体性能达到最新最优水平（截止2025年11月11日），超过第二名的 Operand ensemble（49.56%）四个百分点 ，充分展示了伐谋 Agent 在机器学习自动化与工程智能化方向的卓越表现。
伐谋 Agent 在该任务中排名第一，超越榜单上所有人类，具体解决方案如下：
* 特征工程：融合均值池化与最大池化，以增强语义表征能力。
* 模型架构：采用基于DeBERTa的深度语言模型作为主干网络。
* 训练策略：引入对抗权重扰动以提升模型鲁棒性，并结合五折交叉验证与热启动的线性学习率调度进行多轮优化。
* 模型集成：对五折验证产生的模型进行集成，输出最终的侮辱概率预测。

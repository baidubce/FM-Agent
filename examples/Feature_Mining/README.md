## 问题来源
本问题选自Kaggle竞赛题目American Express - Default Prediction


## 问题描述
信贷违约预测的核心任务是根据客户的信用卡账单、交易行为及账户特征等数据，预测其未来是否违约。该任务数据维度高、时间跨度长且存在较多噪声，对特征工程提出了极高要求，模型需要从复杂的时序金融数据中提取稳定且具有预测价值的特征，以准确刻画客户的信用风险模式。


## 应用场景
信贷违约预测在金融风控领域中具有重要的现实应用价值：
* 信用评估与贷款审批：通过预测客户违约概率，金融机构可优化授信策略、降低坏账率。
* 客户分层与精准营销：依据风险等级制定差异化金融服务，提升资产收益与客户体验。


## 伐谋成果
在信贷违约预测任务中，伐谋 Agent 通过自主生成与进化机制，挖掘出时间序列趋势、波动率、类别主导度、时间衰减等高价值特征，在固定模型和超参数的前提下，将官方评测指标 Amex 从 0.7951 提升至 0.7981，展示了伐谋 Agent 在特征挖掘及特征组合方面的强大能力。
